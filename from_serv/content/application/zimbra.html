Поиск писем

Для поиска требуемой информации в почте необходимо ввести ключевое слово (или слова) в строку поиска.

    При этом:
  • задавая ключевое слово в пустой строке поиска, будет осуществляться поиск во всей почте;
  • после фразы «in:"inbox"» поиск будет осуществляться только среди входящих писем;
  • после фразы «in:"sent"» поиск будет осуществляться только среди исходящих писем.

Сортировка писем

Для сортировки писем необходимо выбрать требуемую папку (например, «Входящие») и нажать столбец, соответствующий критерию поиска (на примере: 1- по отправителю, 2 – по размеру, 3 – по дате получения письма).


Второй щелчок по тому же столбцу изменит порядок следования списка (например, при сортировке по размеру порядок следования писем от большего размера к меньшему измениться на противоположный).

Создание фильтра

Фильтр – это задание некоторого условия, при котором автоматически будет выполняться определенное действие с выбранным письмом/письмами. Например, можно задать условие, при котором все письма, полученные с заданного эл. адреса будут помещаться в определенную выбранную папку.

Для создания нового фильтра выбираем сверху вкладку «Настройки», раздел «Фильтры» и нажимаем кнопку «Новый фильтр»:


Далее задаем имя фильтру и следующие исходные условия: - выбираем критерий писем


- выбираем правило отбора писем


- выбираем выполняемое действие


Сохраняем. Фильтр готов.
